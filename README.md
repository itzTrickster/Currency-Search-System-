# Currency Search System | Система поиска валют

## English

Educational project written in C that demonstrates the use of structures, arrays, string processing, and search algorithms.

### Features
- Search currencies by code
- Search currencies by name
- Display exchange rates relative to RUB
- Store currency data using structures

## Русский

Учебный проект на языке C, демонстрирующий работу со структурами, массивами, обработкой строк и алгоритмами поиска.

### Возможности
- Поиск валют по коду
- Поиск валют по названию
- Отображение курса относительно рубля
- Хранение данных с использованием структур
