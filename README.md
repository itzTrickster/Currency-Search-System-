# Currency Search System | Система поиска валют

## English

Educational project written in C that demonstrates the use of structures, arrays, string processing, and search algorithms. The application allows users to search for currencies by code or name and displays exchange rates relative to the Russian ruble.

## Русский

Учебный проект на языке C, демонстрирующий работу со структурами, массивами, обработкой строк и алгоритмами поиска. Программа позволяет искать валюты по коду или названию и отображает курс относительно российского рубля.
